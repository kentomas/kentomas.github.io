# VS Code

## Keyboard Shortcuts

| Shortcut                    | Description                     |
| --------------------------- | ------------------------------- |
| `Shift` + `Alt` + `K` + `I` | Multi-line selection.           |
| `Shift` + `Alt`+`F`         | Format Document.                |
| `Shift` + `Ctrl` + `P`      | Show Command Pallete.           |
| `Alt` + `Z`                 | Toggle line-wrap.               |
| `Ctrl` + `~`                | Toggle in-built command prompt. |
